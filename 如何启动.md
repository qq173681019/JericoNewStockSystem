# 🚀 快速启动指南 | Quick Start Guide

## 📌 重要提示 | Important Notice

本项目提供了**两个不同的界面**，请根据需要选择：

**This project provides TWO different interfaces:**

---

## 🌐 方式一：Web界面（推荐 | Recommended）

### ✨ 新的现代化Web界面

这是**最新开发的美观Web界面**，具有以下特点：
- 🎨 现代化设计，支持深色/浅色主题
- 📱 响应式布局，支持手机/平板/电脑
- 🚀 无需安装额外GUI库
- 💻 在浏览器中运行

### 启动方法：

#### Windows用户（推荐）：
**双击运行以下文件之一：**
- `启动Web界面.bat` （中文）
- `START_WEB_UI.bat` （English）

#### 或者使用命令行：
```bash
# 1. 安装依赖（首次运行）
pip install Flask Flask-CORS

# 2. 启动Web服务器
python run_web_ui.py

# 3. 浏览器会自动打开 http://127.0.0.1:5000
```

## 🔧 故障排除 | Troubleshooting

### 问题1: 运行main.py看不到新界面
**原因**: `main.py` 启动的是**桌面版**，不是Web界面。

**解决方案**: 
- 运行 `run_web_ui.py` 或双击 `启动Web界面.bat`

### 问题2: 批处理文件报错 "can't open file" 或 "No such file or directory"
**原因**: 批处理文件找不到Python文件

**解决方案**: 
1. **确保你在项目根目录**：打开命令提示符，运行：
   ```bash
   cd C:\Users\你的用户名\Documents\GitHub\JericoNewStockSystem
   ```
2. **或者直接在文件浏览器中双击** `.bat` 文件（现在已修复，会自动切换到正确目录）

### 问题3: 缺少依赖包
**Web界面需要**:
```bash
pip install Flask Flask-CORS
```

**桌面版需要**:
```bash
pip install customtkinter
```

**注意**: 新版本的批处理文件会自动检查并安装缺失的依赖包！

### 问题4: Python命令不存在
**原因**: Python未安装或未添加到系统PATH

**解决方案**:
1. 从 [python.org](https://www.python.org/downloads/) 下载安装Python
2. 安装时勾选 "Add Python to PATH"
3. 重启命令提示符

### 问题5: 端口被占用
如果5000端口被占用，编辑 `run_web_ui.py`，修改：
```python
port = 5000  # 改为其他端口，如 8080
```

### 问题6: 导入错误 (ImportError)
**原因**: 缺少项目依赖

**解决方案**:
```bash
# 安装所有依赖
pip install -r requirements.txt

# 或单独安装Web UI依赖
pip install Flask Flask-CORS
```
