# SIAPS Web UI

现代化的股票智能分析预测系统 Web 界面

## 功能特点

### 🎨 美观的界面设计
- 现代化的响应式设计
- 深色/浅色主题切换
- 流畅的动画效果
- 直观的用户体验

### 📊 核心功能
- **股票预测**: 输入股票代码进行短期、中期、长期预测
- **观测池**: 自定义观察列表，设置目标价格和止损价格
- **历史记录**: 查看所有预测历史和准确率统计
- **技术分析**: 查看技术指标和图表分析

### 💡 前端技术
- 纯 HTML/CSS/JavaScript 实现
- 无需复杂的前端框架
- 使用 LocalStorage 进行数据持久化
- Font Awesome 图标库

### 🔧 后端技术
- Flask 轻量级 Web 框架
- 与现有 Python 代码集成
- RESTful API 设计

## 快速开始

### 1. 安装依赖

```bash
pip install flask>=3.0.0
```

或者使用完整的 requirements.txt:

```bash
pip install -r requirements.txt
```

### 2. 启动 Web 服务器

```bash
python web_ui.py
```

### 3. 打开浏览器

访问: http://127.0.0.1:5000

## 使用说明

### 股票预测
1. 点击左侧导航栏的"股票预测"
2. 输入6位股票代码（例如: 000001）
3. 选择预测类型（短期/中期/长期）
4. 点击"开始预测"按钮
5. 查看预测结果和建议

### 观测池管理
1. 点击左侧导航栏的"观测池"
2. 点击右上角"添加股票"按钮
3. 填写股票信息和目标价格
4. 点击"添加"保存到观测池
5. 可以随时查看和管理观测池中的股票

### 历史记录
1. 点击左侧导航栏的"历史记录"
2. 查看所有预测记录
3. 点击"查看"按钮查看详细信息
4. 使用筛选器按时间范围查看

### 主题切换
- 点击左侧边栏底部的主题切换开关
- 在深色和浅色主题之间切换
- 设置会自动保存到浏览器

## API 端点

### GET /
返回主页面

### GET /api/info
获取应用信息
```json
{
  "name": "SIAPS",
  "version": "0.1.0",
  "status": "running"
}
```

### POST /api/validate_stock
验证股票代码
```json
{
  "code": "000001"
}
```

### POST /api/predict
执行股票预测（占位符，待集成 ML 模型）
```json
{
  "code": "000001",
  "type": "short"
}
```

## 数据存储

- 观测池和历史记录存储在浏览器的 LocalStorage 中
- 数据在浏览器端持久化
- 清除浏览器缓存会删除数据
- 未来版本将支持服务器端存储
