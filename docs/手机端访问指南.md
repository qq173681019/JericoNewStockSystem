# 📱 手机端访问指南 | Mobile Access Guide

本指南将帮助您通过手机访问 SIAPS 股票智能分析预测系统的 Web 界面。

This guide will help you access the SIAPS Stock Analysis System from your mobile phone.

---

## 🚀 快速启动 | Quick Start

### 方法一：使用手机版启动脚本（推荐）

#### Windows 用户：
1. 双击运行 **`启动Web界面-手机版.bat`**
2. 等待服务器启动，记下显示的访问地址（例如：http://192.168.1.100:5000）
3. 在手机浏览器中输入该地址

### 方法二：使用命令行参数

```bash
python run_web_ui.py --mobile
```

或者指定自定义端口：

```bash
python run_web_ui.py --mobile --port 8080
```

---

## 📋 详细步骤 | Detailed Steps

### 步骤 1：启动服务器

有三种方式启动支持手机访问的服务器：

#### 选项 A：双击批处理文件（最简单）
- 双击运行 `启动Web界面-手机版.bat`

#### 选项 B：使用 --mobile 参数
```bash
python run_web_ui.py --mobile
```

#### 选项 C：使用 --host 参数
```bash
python run_web_ui.py --host 0.0.0.0
```

### 步骤 2：查看访问地址

服务器启动后，会显示类似如下信息：

```
================================================================
  SIAPS - 股票智能分析预测系统 Web UI
  Stock Intelligent Analysis & Prediction System
================================================================

🌐 网络访问模式 (Network Access Mode)
📱 手机访问 (Mobile Access): http://192.168.1.100:5000
💻 本地访问 (Local Access): http://127.0.0.1:5000

⚠️  确保您的手机和电脑在同一局域网内
⚠️  Make sure your phone and computer are on the same network
```

**重要**：记下 "📱 手机访问" 后面显示的地址（例如：`http://192.168.1.100:5000`）

### 步骤 3：连接到同一网络

确保您的手机和运行服务器的电脑连接到**同一个 WiFi 网络**。

### 步骤 4：在手机上访问

1. 打开手机上的浏览器（推荐使用 Chrome、Safari 或 Edge）
2. 在地址栏输入步骤 2 中记下的地址
3. 按回车访问

## 🔧 命令行参数说明 | Command Line Options

| 参数 | 说明 | 示例 |
|------|------|------|
| `--mobile` | 启用手机访问模式（自动设置 host 为 0.0.0.0） | `python run_web_ui.py --mobile` |
| `--host` | 指定主机地址 | `python run_web_ui.py --host 0.0.0.0` |
| `--port` | 指定端口号（默认 5000） | `python run_web_ui.py --port 8080` |
| `--no-browser` | 不自动打开浏览器 | `python run_web_ui.py --no-browser` |

### 参数组合示例：

```bash
# 手机访问，使用 8080 端口，不打开浏览器
python run_web_ui.py --mobile --port 8080 --no-browser

# 允许网络访问，使用 5001 端口
python run_web_ui.py --host 0.0.0.0 --port 5001
```

## ❓ 常见问题 | FAQ

### Q1: 为什么手机无法访问？

**答：** 请检查以下几点：
1. 手机和电脑是否连接到同一个 WiFi 网络
2. 是否使用了正确的 IP 地址（不是 127.0.0.1）
3. 防火墙是否阻止了端口 5000（Windows 防火墙可能需要允许访问）
4. 服务器是否使用了 `--mobile` 参数或 `--host 0.0.0.0` 启动

### Q2: 如何找到我的电脑 IP 地址？

**答：** 
- **Windows**: 打开命令提示符，输入 `ipconfig`，查找 "IPv4 地址"
- **Mac/Linux**: 打开终端，输入 `ifconfig` 或 `ip addr`
- **使用本程序**: 服务器启动时会自动显示 IP 地址

### Q3: 端口 5000 被占用怎么办？

**答：** 使用 `--port` 参数指定其他端口：
```bash
python run_web_ui.py --mobile --port 8080
```

### Q4: 可以同时让多个手机访问吗？

**答：** 可以！多个设备可以同时访问同一个服务器。

### Q5: 访问速度慢怎么办？

**答：** 
- 确保使用的是 5GHz WiFi（如果可用）
- 尝试靠近路由器
- 检查网络连接质量
- 如果数据量大，考虑在电脑上直接访问

### Q6: 如何在外网访问？

**答：** 
- 本系统默认只支持局域网访问，这是出于安全考虑
- 如需外网访问，您需要：
  1. 配置路由器端口转发
  2. 使用 VPN
  3. 使用内网穿透工具（如 ngrok、frp 等）
  
  **注意**：外网访问可能带来安全风险，请谨慎操作。

### Q7: Windows 防火墙提示怎么办？

**答：** 
- 第一次运行时，Windows 可能会弹出防火墙警告
- 请选择 "允许访问" 或 "Allow access"
- 如果不小心拒绝了，可以在 Windows 防火墙设置中手动添加例外

## 🔒 安全提示 | Security Notes

1. **局域网访问**：默认情况下，服务器仅在局域网内可访问，这是安全的。
2. **不要暴露到公网**：除非您了解相关风险并采取了适当的安全措施。
3. **生产环境**：如果在生产环境使用，请更改 `app.config['SECRET_KEY']`。
4. **防火墙**：确保防火墙规则正确配置，只允许可信设备访问。
