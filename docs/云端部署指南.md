## 🚀 快速部署到 Vercel

### 方法一：一键部署（推荐）

1. **点击部署按钮**：

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/qq173681019/JericoNewStockSystem)

2. **完成设置**：
   - 使用 GitHub 账号登录 Vercel
   - 点击 "Create" 创建项目
   - 等待 2-3 分钟完成部署

3. **获取访问地址**：
   - 部署完成后会自动生成域名
   - 类似 `your-project.vercel.app`
   - 在手机浏览器输入此域名即可访问

### 方法二：从 GitHub 导入

1. **准备 GitHub 仓库**：
   - Fork 或 Clone 本项目到您的 GitHub
   - 确保代码已推送到 GitHub

2. **访问 Vercel**：
   - 打开 https://vercel.com/
   - 使用 GitHub 账号登录
   - 点击 "Add New Project"

3. **导入项目**：
   - 选择 "Import Git Repository"
   - 授权 Vercel 访问您的 GitHub
   - 选择 `JericoNewStockSystem` 仓库
   - 点击 "Import"

4. **配置项目**（可选）：
   - Framework Preset: 选择 "Other"
   - Build Command: 留空
   - Output Directory: 留空
   - Install Command: 使用默认 `pip install -r requirements.txt`

5. **部署**：
   - 点击 "Deploy"
   - 等待 2-3 分钟完成部署
   - 部署完成后获取访问地址

## 📱 在手机上使用

### 1. 打开浏览器访问

在手机浏览器中输入您的 Vercel 域名：
```
https://your-project.vercel.app
```

### 2. 添加到主屏幕（推荐）

**iOS (Safari)**:
1. 点击底部分享按钮
2. 选择"添加到主屏幕"
3. 设置名称为 "股票分析"
4. 点击"添加"

**Android (Chrome)**:
1. 点击菜单按钮（三个点）
2. 选择"添加到主屏幕"
3. 设置名称为 "股票分析"
4. 点击"添加"

现在您可以像使用 App 一样使用它！

## 🔧 常见问题

### Q1: 部署失败怎么办？

**A**: 检查以下几点：
1. 查看 Vercel 部署日志是否有错误
2. 确认 `vercel.json` 文件存在且正确
3. 确认 `requirements.txt` 中的依赖都是轻量级的（不包含 PyTorch、TensorFlow 等大型库）

### Q2: 数据会保存吗？

**A**: 不会。Vercel 是无状态的 serverless 平台。
- SQLite 数据库会在每次部署后重置
- 观察列表等数据不会持久化
- 如需持久化数据，考虑：
  - 使用外部数据库（PostgreSQL、MySQL）
  - 使用 Vercel KV 存储
  - 使用浏览器本地存储

### Q3: 有访问限制吗？

**A**: 免费版限制：
- 100 GB 流量/月（个人使用足够）
- 函数执行时间最长 10 秒
- 如需更多，可升级到 Pro 计划

### Q4: 可以自定义域名吗？

**A**: 可以！
1. 在 Vercel 项目设置中点击 "Domains"
2. 添加您的域名
3. 按照提示配置 DNS
4. 等待 DNS 生效（通常几分钟到几小时）

### Q5: 部署后访问很慢怎么办？

**A**: Vercel 使用全球 CDN，应该很快。如果慢可能是：
- 首次访问需要冷启动（等待 5-10 秒）
- 网络问题
- 依赖包过大导致加载慢

## ⚠️ 注意事项

1. **不要在生产环境使用敏感数据**：Vercel 函数日志可能包含请求信息
2. **定期检查使用量**：确保不超过免费额度
3. **代码变更会触发重新部署**：每次 Git Push 都会重新部署
4. **环境变量在 Vercel 中设置**：不要在代码中硬编码敏感信息
