# 手机版本如何运行

## 🎯 最简单的方法

### Windows 用户
直接双击运行这个文件：
```
启动Web界面-手机版.bat
```

### 命令行用户
```bash
python run_web_ui.py --mobile
```

---

## 📱 完整步骤

### 第一步：启动服务器

在电脑上运行以下命令之一：

**方法1：双击批处理文件（推荐）**
- 找到 `启动Web界面-手机版.bat` 文件
- 双击运行

**方法2：命令行**
```bash
cd 到项目目录
python run_web_ui.py --mobile
```

### 第二步：查看IP地址

启动后，屏幕上会显示：
```
================================================================
  SIAPS - 股票智能分析预测系统 Web UI
================================================================

🌐 网络访问模式 (Network Access Mode)
📱 手机访问 (Mobile Access): http://192.168.1.100:5000
💻 本地访问 (Local Access): http://127.0.0.1:5000
```

记下 "📱 手机访问" 后面的地址，例如：`http://192.168.1.100:5000`

### 第三步：连接WiFi

确保您的**手机**和**电脑**连接到**同一个WiFi网络**。

### 第四步：手机访问

1. 打开手机浏览器（推荐Chrome或Safari）
2. 在地址栏输入第二步记下的地址
3. 按回车访问

---

## 💡 其他选项

### 修改端口号
如果5000端口被占用：
```bash
python run_web_ui.py --mobile --port 8080
```

### 不打开浏览器
```bash
python run_web_ui.py --mobile --no-browser
```

### 查看帮助
```bash
python run_web_ui.py --help
```

---

## ❓ 常见问题

### 手机无法访问？
1. 检查手机和电脑是否在同一WiFi
2. 检查是否使用了正确的IP地址
3. 检查防火墙是否阻止了端口5000
4. 确认使用了 `--mobile` 参数启动

### 如何找到电脑的IP地址？
- **Windows**: 命令提示符输入 `ipconfig`
- **Mac/Linux**: 终端输入 `ifconfig`
- **自动显示**: 使用 `--mobile` 启动时会自动显示

### 端口被占用？
使用不同的端口：
```bash
python run_web_ui.py --mobile --port 8080
```

---

## 🔒 安全提示

⚠️ **重要**: 手机访问模式会将服务暴露到网络
- 仅在家庭或公司等可信任的WiFi网络中使用
- 不要在公共WiFi中使用
- 不要将服务暴露到互联网
