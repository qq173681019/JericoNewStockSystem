# SIAPS Web UI - 股票智能分析预测系统

一个现代化、美观的Web界面，用于股票智能分析和预测。

## 🎨 界面特性

### 设计理念
- **现代化设计**: 采用流行的扁平化设计风格，界面简洁美观
- **响应式布局**: 完美支持桌面端、平板和移动端设备
- **深色/浅色主题**: 支持主题切换，保护用户视力
- **流畅动画**: 精心设计的过渡动画，提供流畅的用户体验
- **直观交互**: 简单易用的交互设计，降低学习成本

### 核心功能

#### 1. 股票预测 📈
- 输入股票代码，获取智能分析结果
- 短期预测（1天涨跌预测）
- 中期预测（3个月目标价位）
- 操作建议（买入/卖出/持有）
- 技术指标显示（RSI、MACD、KDJ、MA等）
- 价格走势图表可视化

#### 2. 观测池管理 👁️
- 添加关注的股票到观测池
- 实时查看股票价格和涨跌幅
- 设置目标价格和预警
- 快速操作（查看详情、删除）

#### 3. 历史记录 📝
- 查看所有预测历史记录
- 统计准确率数据
- 按时间筛选记录
- 对比预测结果和实际结果

#### 4. 数据分析 📊
- 板块热度分析
- 市场情绪指标
- 可视化图表展示

#### 5. 系统设置 ⚙️
- 通知设置
- 数据源配置
- 系统信息

## 🚀 快速启动

### 前置要求
- Python 3.8+
- 已安装项目依赖

### 安装依赖
```bash
pip install Flask Flask-CORS
```

### 启动Web UI
```bash
python run_web_ui.py
```

启动后：
1. 浏览器会自动打开
2. 或手动访问 http://127.0.0.1:5000
3. 开始使用系统进行股票分析

## 🎯 技术栈

### 前端
- **HTML5**: 语义化标签，结构清晰
- **CSS3**: 现代CSS特性（Grid、Flexbox、CSS Variables）
- **JavaScript (ES6+)**: 原生JavaScript，无框架依赖
- **Canvas API**: 原生图表绘制，无需外部库

### 后端
- **Flask**: 轻量级Python Web框架
- **Flask-CORS**: 跨域资源共享支持

## 💡 使用说明

### 进行股票预测
1. 在导航栏点击"股票预测"
2. 在输入框输入股票代码（例如：000001）
3. 点击"开始预测"按钮
4. 等待分析完成，查看结果

### 管理观测池
1. 在导航栏点击"观测池"
2. 点击"添加股票"按钮
3. 输入股票代码
4. 查看观测池中的股票实时数据

### 查看历史记录
1. 在导航栏点击"历史记录"
2. 查看预测统计数据
3. 浏览历史预测记录
4. 使用筛选器过滤数据

### 切换主题
- 在侧边栏底部找到主题切换开关
- 切换深色/浅色模式
- 系统会记住您的偏好设置

## 🔧 配置说明

### 修改服务器端口
编辑 `run_web_ui.py` 文件：
```python
port = 5000  # 修改为您想要的端口
```

### 自定义主题颜色
编辑 `web_ui/static/css/style.css` 的 CSS 变量：
```css
:root {
    --primary-color: #3b82f6;  /* 主色调 */
    --success-color: #10b981;  /* 成功色 */
    /* 更多颜色变量... */
}
```

## 📝 开发说明

### 添加新的视图
1. 在 `index.html` 中添加新的 `<div class="view">` 
2. 在导航菜单中添加对应的 `<li class="nav-item">`
3. 在 `app.js` 中添加视图切换逻辑

### 添加新的API端点
1. 在 `run_web_ui.py` 中添加新的路由
2. 在前端 `app.js` 中调用API

### 自定义样式
- 所有颜色通过CSS变量定义，方便主题定制
- 响应式断点可在CSS中调整
- 支持CSS Grid和Flexbox布局

## 🐛 故障排除

### 问题：端口被占用
**解决方案**: 修改 `run_web_ui.py` 中的端口号

### 问题：样式不显示
**解决方案**: 检查静态文件路径配置，清除浏览器缓存

### 问题：图表不显示
**解决方案**: 图表使用原生Canvas绘制。检查浏览器控制台是否有JavaScript错误
